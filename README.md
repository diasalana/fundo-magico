# Background Generator

Projeto para gerar código HTML e CSS a partir da descrição de um background informada pelo usuário.

O objetivo é facilitar a criação de estilos visuais, automatizando a geração de uma base de layout conforme o contexto descrito.

## Objetivo

- Gerar estruturas básicas de HTML e CSS
- Criar backgrounds a partir de descrições
- Acelerar a prototipação de layouts
- Padronizar a criação de estilos visuais

## Tecnologias

- HTML5  
- CSS3  
- JavaScript  
- n8n  

## Funcionamento

1. O usuário descreve o background desejado
2. O sistema processa a descrição
3. O projeto gera:
   - Estrutura HTML
   - Estilos CSS correspondentes
